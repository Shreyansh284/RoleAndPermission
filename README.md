# **Role Management System**  

## **Overview**  
This is a Role Management System built using Laravel. It allows administrators to manage roles and permissions dynamically through a user-friendly interface.

---

## **Features**  
- Create, Read, Update, and Delete (CRUD) roles.  
- Assign permissions to roles.  
- Update permissions through checkboxes.  
- Responsive and modern UI using Bootstrap.
